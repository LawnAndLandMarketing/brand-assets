# Lawn & Land Marketing — Brand Assets

Permanent public CDN for L&L brand assets. Raw file URLs are stable and can be used directly in any project.

## Usage

```
https://raw.githubusercontent.com/LawnAndLandMarketing/brand-assets/main/logos/[filename]
```

## Logo Files

| File | Description |
|------|-------------|
| `horizontal.png` | Horizontal logo — full color (default) |
| `horizontal-black.png` | Horizontal logo — black |
| `horizontal-white.png` | Horizontal logo — white |
| `badge.png` | Badge/icon — full color |
| `badge-black.png` | Badge/icon — black |
| `badge-white.png` | Badge/icon — white |
| `vertical.png` | Vertical stacked — full color |
| `vertical-black.png` | Vertical stacked — black |
| `vertical-white.png` | Vertical stacked — white |
| `horizontal-bordered-thin.png` | Horizontal with thin border |
| `horizontal-bordered-thick.png` | Horizontal with thick border |
| `badge-bordered-thin.png` | Badge with thin border |
| `badge-bordered-thick.png` | Badge with thick border |
| `vertical-bordered-thin.png` | Vertical with thin border |
| `vertical-bordered-thick.png` | Vertical with thick border |
